# Six Sigma Jupyter Notebooks

This repository contains Jupyter notebooks for Six Sigma procedures, based on material from the book *Six Sigma: A Case Study Approach Using Minitab*. This is a test for branch, commit and push. This is a second test for new branch. This is a third test.

## Overview

The Six Sigma methodology is used to improve processes by identifying and removing causes of defects. These notebooks cover key Six Sigma stages, including Define, Measure, Analyze, Improve, and Control (DMAIC).

## Notebooks

- **Define**: Description of the Define stage and how it is applied in Six Sigma.
- **Measure**: Measuring process performance and data collection techniques.
- **Analyze**: Analyzing the data and identifying root causes.
- **Improve**: Techniques for process improvement.
- **Control**: Control charts and process monitoring.

## Requirements

- Python 3.x
- Jupyter Notebook
- Libraries: `numpy`, `pandas`, `matplotlib`, `seaborn`, etc.
